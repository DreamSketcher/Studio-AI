## 2. Новые компоненты

### `ai_studio_core/espeak_tts.py` (ядро)
- `find_espeak()/available()` — поиск `espeak-ng`/`espeak`.
- `synthesize(text, out_path, language, speed)` → реальный WAV 22050 Гц mono 16-bit + факты (duration/frames/rate).
- `resolve_voice` — автоопределение ru/en по кириллице; `wpm_from_speed` — скорость UI→WPM.
- Отказ бэкенда → `RuntimeError`, **файл-пустышка не создаётся**.

### `TTSController` — переписан
- Автовыбор бэкенда: `coqui` (если torch+TTS) → `espeak` → честная ошибка `ctl_tts_missing`.
- Цепочка: normalize → chunk → синтез по чанкам → склейка `wave` (без внешних зависимостей, 150 мс паузы) → имя через `output_naming` → конвертация формата (pydub+ffmpeg) → `save_history` → сигналы.
- Coqui-путь реализован полностью (XTTS v2 требует reference-аудио — без него честная ошибка с инструкцией), в этой среде не тестируется (нет torch) — выбор бэкенда автоматический.
- espeak честно логирует: reference audio игнорируется (клонирование голоса — только Coqui).
- Интеграция с очередью через `attach_queue(qc)`; прогресс/статусы реальные.
- `export_last(target)` — копия/конвертация с валидацией.

### `HistoryPanel` — переписана
`scan_outputs(OUTPUT_DIR)`: реальные файлы (`wav/mp3/flac/ogg`), сортировка по mtime, имя/дата/размер, поиск, выбор → `item_selected(path)` → Inspector показывает Size/Path, двойной клик — тот же сигнал. Пустое состояние — честный текст, не манекены.

### `MainWindow`
- `_on_tts_done`: волна из реального файла, Export активна, autoplay (если включён в сайдбаре), `history.refresh()`.
- `_on_export`: диалог сохранения → `export_last` → toast/лог; «нечего экспортировать» — честное предупреждение.

## 3. Тесты (новые, 46 шт.)

**`test_espeak_backend.py`** (15): обнаружение бинарника; автоопределение языка;
границы WPM; **реальный синтез EN/RU** (файл, 22050 Гц, >1000 сэмплов,
размер ≥ полусекунды речи — анти-пустышка); перезапись в явный путь;
пустой текст → ValueError; отсутствие бинарника → RuntimeError без файла.

**`test_waveform_real.py`** (10): пики синуса (форма, нормировка), ступенька
амплитуды, zero-length → ValueError, пики реальной речи (есть паузы),
длительность, формат времени, виджет offscreen (пики/время/путь),
кнопка play честно отключена без мультимедиа-бэкенда.

**`test_tts_flow_real.py`** (15): бэкенд здесь = espeak; полный поток с
реальным WAV (>20 KB, 22050 Гц); pipeline-steps доходят до Output;
мульти-чанк склейка (длительность >3 с); вывод mp3 через ffmpeg real;
запись в history.json; RVC-гейт (ошибка, генерации НЕТ); отказ при
недоступном бэкенде; очередь стартует пустой (анти-демо), жизненный цикл
задачи done/100, cancel, clear; экспорт-копия байт-в-байт, экспорт без
генерации → RuntimeError, цель без расширения → ValueError;
отмена → задача cancelled.

**`test_history_real.py`** (6): скан только аудио sorted desc; нет каталога → [];
пусто → честный текст; выбор → реальный путь; поиск-фильтр; refresh подхватывает новый файл.
